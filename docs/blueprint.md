# **App Name**: ProCalc Hub

## Core Features:

- Calculator Pages: Dedicated pages for each calculator with interactive UI, result display, disclaimer, related calculators, article section, and ad placeholders.
- Diverse Calculator Suite: Implement a suite of calculators covering mathematics, finance, health, and conversions, ensuring decimal-safe arithmetic.
- Search and Discovery: Enable global search with autocomplete to quickly find calculators and curate featured calculators on the home page.
- User Authentication and History: Implement user authentication with social providers and email, automatically saving calculation history for logged-in users.
- Appearance Customization: Offer global appearance settings with light, dark, dim, and custom color options, saved locally and to user preferences.
- Formula Assistance: Provide AI powered help, explaining the formulae used in the different calculators. User input history should not be retained. This tool would enhance understanding and user confidence in results. A switch is exposed to enable / disable it
- Ad Readiness and Analytics: Include ad placeholders with clear instructions for AdSense/AdMob integration, along with Google Analytics setup instructions.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey professionalism and trust.
- Background color: Light Gray (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#009688) for highlighting key elements and CTAs.
- Body and headline font: 'Inter', a sans-serif font providing a modern, machined, objective, and neutral feel for both headlines and body text.
- Use minimalistic, clear icons for categories and settings.
- Maintain a clean, well-spaced layout with a focus on keyboard navigation and mobile-friendliness.
- Use subtle micro-interactions and transitions for a polished user experience.
